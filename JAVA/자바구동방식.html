# 자바의 구동 방식

-- 작성이 완료된 글이 아닙니다.

자바의 구동 방식에 대해 자세히 설명하기 전에 간단하게(?) 한 줄로 이야기 해보자면, 다음과 같다.

개발자가 작성한 소스코드(.java)를 컴파일러(javac.exe)가 바이트코드(.class 파일)로 컴파일하고 런처(java.exe)로 자바 가상 머신(JVM)을 구동시켜 컴파일된 바이트 코드를 해석하여 자바 프로그램이 실행된다.

밑에서 더 자세히 설명을 해보도록 하겠다.
